Entity 架构再深入：它到底是什么？

核心

🎨 一句话定义

Entity 是 CesiumJS 中描述"地球上某个对象"的高级抽象。一个 Entity 可以包含位置、朝向、一个或多个图形（Graphics），以及随时间变化的属性。它相当于你在三维场景里的"演员"——每个演员有自己的姿态、服装和动作。

你通过 viewer.entities.add(entityConfig) 创建它，CesiumJS 会自动把它转换为底层的 Primitive 进行渲染。

🛠️ 一个 Entity 的完整结构

viewer.entities.add({
  id:          'entity-01',          // 唯一标识
  name:        '无人机 A',            // 名称
  description: '...',                // HTML 描述（点击信息框显示）
  position:    Cesium.Cartesian3.fromDegrees(116.40, 39.90, 100),
  orientation: Cesium.Transforms.headingPitchRollQuaternion(
                   Cesium.Cartesian3.fromDegrees(116.40, 39.90, 100),
                   new Cesium.HeadingPitchRoll(0, 0, 0)
                 ),
  viewFrom:    new Cesium.Cartesian3(-100, 0, 50), // 跟随视角
  availability: new Cesium.TimeIntervalCollection([{
                   start: Cesium.JulianDate.fromDate(new Date('2026-01-01')),
                   stop:  Cesium.JulianDate.fromDate(new Date('2026-01-02'))
                 }]),
  // 图形：一个 Entity 可以同时挂多个图形
  point:   { pixelSize: 10, color: Cesium.Color.RED },
  label:   { text: '无人机 A' },
  polyline:{ positions: [...], width: 2 },
  model:   { uri: 'uav.glb' }
});
💡
Entity 不是 Primitive：Entity 是"高级 API"，Primitive 是"底层 API"。Entity 更易用、可读性更好；Primitive 性能更高。日常工作 90% 用 Entity 即可，只有遇到 10 万+ 海量对象时才需要 Primitive。

点、图标、标签：最常见三位一体

0D

🔴 Point — 像素点

Point 是最简单的图形，适合标记位置、传感器、小目标。

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.3974, 39.9093),
  point: {
    pixelSize: 16,
    color: Cesium.Color.RED,
    outlineColor: Cesium.Color.WHITE,
    outlineWidth: 2,
    scaleByDistance: new Cesium.NearFarScalar(1.5e2, 20, 1.5e7, 1) // 近大远小
  }
});

📌 Billboard — 图片图标

Billboard 在地球上贴一张图片，常用于 POI 图标、车辆图标、飞机图标。支持缩放、旋转、透明、对齐方式。

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.3974, 39.9093, 100),
  billboard: {
    image: 'pin.png',            // 图片 URL 或 Data URI
    scale: 1.0,
    color: Cesium.Color.WHITE,
    verticalOrigin: Cesium.VerticalOrigin.BOTTOM, // 底部对齐地面
    horizontalOrigin: Cesium.HorizontalOrigin.CENTER,
    pixelOffset: new Cesium.Cartesian2(0, -10),
    rotation: Cesium.Math.toRadians(30), // 旋转角度
    alignedAxis: Cesium.Cartesian3.ZERO // 随视角旋转
  }
});

💬 Label — 文字标签

Label 在地球上显示文字，通常和 Point/Billboard 组合使用。

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.3974, 39.9093, 120),
  label: {
    text: '北京',
    font: '18px sans-serif',
    fillColor: Cesium.Color.WHITE,
    outlineColor: Cesium.Color.BLACK,
    outlineWidth: 3,
    style: Cesium.LabelStyle.FILL_AND_OUTLINE,
    verticalOrigin: Cesium.VerticalOrigin.BOTTOM,
    pixelOffset: new Cesium.Cartesian2(0, -24)
  }
});
💡
三者组合：最常见的标注模式是 billboard + label——图标 + 文字。Point 适合做"小圆点"状态指示，如实时目标位置。
4

线、走廊、墙：一维图形的空间延伸

1D

➔ Polyline — 折线

Polyline 由多个点连接而成，适合航线、轨迹、边界。可以设置宽度、颜色、虚线、发光效果。

viewer.entities.add({
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([
      116.39, 39.91,   // 北京
      121.47, 31.23,   // 上海
      113.26, 23.13    // 广州
    ]),
    width: 4,
    material: new Cesium.PolylineGlowMaterialProperty({
      glowPower: 0.2,
      color: Cesium.Color.CYAN
    }),
    clampToGround: true // 贴地（沿地形起伏）
  }
});

↔ Corridor — 带状走廊

Corridor 像一条有宽度的道路，会自动贴合地形。适合做道路、航线缓冲带、河流。

viewer.entities.add({
  corridor: {
    positions: Cesium.Cartesian3.fromDegreesArray([
      116.39, 39.91,
      121.47, 31.23
    ]),
    width: 20000, // 宽度 20km
    material: Cesium.Color.ORANGE.withAlpha(0.5),
    outline: true,
    outlineColor: Cesium.Color.BLACK
  }
});

🧭 Wall — 垂直墙

Wall 从地面向上拉起一面墙，常用于围栏、雷达遮挡、可视化区域。

viewer.entities.add({
  wall: {
    positions: Cesium.Cartesian3.fromDegreesArray([
      116.0, 39.5,
      117.0, 39.5,
      117.0, 40.5,
      116.0, 40.5
    ]),
    maximumHeights: [10000, 10000, 10000, 10000], // 每点墙高
    minimumHeights: [0, 0, 0, 0],
    material: Cesium.Color.RED.withAlpha(0.4)
  }
});
💡
Polyline 贴地 vs 空中：设置 clampToGround: true 后，线会沿着地形起伏；否则是直线连接起点和终点，可能穿过山脉。轨迹展示通常用 clampToGround，飞行航线通常不用。
5

面、椭圆、矩形：二维图形的覆盖

2D

◻ Polygon — 多边形

Polygon 由闭合边界围成，可填充材质、显示轮廓。支持挖洞（holes）和拉伸高度（extrudedHeight）。

viewer.entities.add({
  polygon: {
    hierarchy: Cesium.Cartesian3.fromDegreesArray([
      116.0, 39.5,
      117.0, 39.5,
      117.0, 40.5,
      116.0, 40.5
    ]),
    material: Cesium.Color.RED.withAlpha(0.3),
    outline: true,
    outlineColor: Cesium.Color.RED,
    extrudedHeight: 5000, // 向上拉伸 5000 米，形成立体建筑
    perPositionHeight: false // 所有点统一高度
  }
});

⚫ Ellipse — 椭圆与圆

Ellipse 以 position 为中心，按 semiMajorAxis / semiMinorAxis 绘制椭圆，适合表示雷达覆盖、影响范围、圆形区域。

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.3974, 39.9093),
  ellipse: {
    semiMajorAxis: 50000, // 长半轴 50km
    semiMinorAxis: 30000, // 短半轴 30km
    rotation: Cesium.Math.toRadians(45), // 旋转 45°
    material: Cesium.Color.BLUE.withAlpha(0.3),
    outline: true,
    outlineColor: Cesium.Color.BLUE
  }
});

☐ Rectangle — 矩形

Rectangle 用西南角和东北角定义一个矩形，适合表示地图视窗、经纬度网格、区域。

viewer.entities.add({
  rectangle: {
    coordinates: Cesium.Rectangle.fromDegrees(115.0, 39.0, 118.0, 41.0),
    material: Cesium.Color.GREEN.withAlpha(0.2),
    outline: true,
    outlineColor: Cesium.Color.GREEN,
    height: 0,
    extrudedHeight: 2000
  }
});
6

体：Box、Cylinder、Ellipsoid、Plane

3D

📦 Box — 立方体

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.40, 39.90, 100),
  box: {
    dimensions: new Cesium.Cartesian3(1000, 500, 300), // 长、宽、高（米）
    material: Cesium.Color.RED.withAlpha(0.5),
    outline: true
  }
});

🛠 Cylinder — 圆柱/圆锥

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.41, 39.91, 0),
  cylinder: {
    length: 2000,         // 高度
    topRadius: 500,      // 顶半径（0=圆锥）
    bottomRadius: 1000, // 底半径
    material: Cesium.Color.ORANGE
  }
});

🔘 Ellipsoid — 椭球/球

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.42, 39.92, 500),
  ellipsoid: {
    radii: new Cesium.Cartesian3(1000, 1000, 1000), // 球
    material: Cesium.Color.BLUE.withAlpha(0.5)
  }
});

🧭 Plane — 平面

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.43, 39.93, 200),
  plane: {
    plane: new Cesium.Plane(0.0, 0.0, 1.0, 0.0), // ax+by+cz+d=0
    dimensions: new Cesium.Cartesian2(1000, 1000)
  }
});
💡
3D 图形的方向：Box 和 Cylinder 默认沿地轴方向，如果需要让模型"立在地面上"，通常需要设置 orientation（四元数）。真实场景中 3D 建筑/设备更常用 model 图形加载 glTF/glb 模型，而不是原生几何体。

材质与外观：让 Entity 更真实

材质

🎨 常用材质类型

材质类名效果
纯色ColorMaterialProperty最常用
图片ImageMaterialProperty贴图
棋盘格CheckerboardMaterialProperty测试用
条纹StripeMaterialProperty斑马纹
网格GridMaterialProperty网格线
发光线PolylineGlowMaterialProperty光晕折线
箭头线PolylineArrowMaterialProperty带方向箭头

🖼 代码示例

// 纯色半透明
material: Cesium.Color.RED.withAlpha(0.3)

// 图片贴图
material: new Cesium.ImageMaterialProperty({
  image: 'radar.png',
  transparent: true
})

// 棋盘格
material: new Cesium.CheckerboardMaterialProperty({
  evenColor: Cesium.Color.WHITE,
  oddColor: Cesium.Color.BLUE,
  repeat: new Cesium.Cartesian2(4, 4)
})

// 发光折线
material: new Cesium.PolylineGlowMaterialProperty({
  color: Cesium.Color.CYAN,
  glowPower: 0.3
})
💡
颜色：Cesium.Color.fromCssColorString('#61dafb') 可以从 CSS 颜色字符串创建 Cesium 颜色；withAlpha(0.5) 设置透明度，这是 2D/3D 覆盖区域最常用的做法。

动态属性：CallbackProperty 与 SampledProperty

时间

⏰ CallbackProperty — 实时计算属性

当属性需要每帧实时计算时（比如随鼠标位置变化、实时位置更新），使用 CallbackProperty。

let lat = 39.90;
let lng = 116.40;

viewer.entities.add({
  position: new Cesium.CallbackProperty(() => {
    // 每帧返回新的位置
    return Cesium.Cartesian3.fromDegrees(lng, lat, 100);
  }, false), // false 表示不是常量（会每帧调用）
  point: { pixelSize: 12, color: Cesium.Color.RED }
});

// 之后更新 lat/lng 即可移动点
setInterval(() => {
  lat += 0.001;
  lng += 0.001;
}, 100);

🕐 SampledProperty — 随时间变化的属性

SampledProperty 用于时间轴动画——你给出几个时刻的值，CesiumJS 自动插值。

const position = new Cesium.SampledPositionProperty();

position.addSample(
  Cesium.JulianDate.fromDate(new Date('2026-01-01T00:00:00Z')),
  Cesium.Cartesian3.fromDegrees(116.40, 39.90, 1000)
);
position.addSample(
  Cesium.JulianDate.fromDate(new Date('2026-01-01T01:00:00Z')),
  Cesium.Cartesian3.fromDegrees(121.47, 31.23, 1000)
);

viewer.entities.add({
  position: position,
  path: { leadTime: 0, trailTime: 3600, width: 3, resolution: 60 },
  model: { uri: 'plane.glb', minimumPixelSize: 64 }
});

viewer.clock.shouldAnimate = true;
💡
动态属性的本质：它们把属性从"静态值"变成"随时间/帧变化的函数"。这是 CesiumJS 做卫星轨迹、飞机航线、车辆追踪、灾害扩散模拟的核心能力。

EntityCollection：批量管理实体

管理

📁 常用 API

// 添加
const entity = viewer.entities.add({ ... });

// 根据 id 获取
const found = viewer.entities.getById('entity-01');

// 根据 id 或 entity 对象移除
viewer.entities.remove(entity);
viewer.entities.removeById('entity-01');

// 移除全部
viewer.entities.removeAll();

// 遍历所有实体
viewer.entities.values.forEach(e => {
  console.log(e.id, e.name);
});

📐 封装：城市标注管理器

class CityMarkerManager {
  constructor(viewer) {
    this.viewer = viewer;
  }

  add(id, name, lng, lat, color = Cesium.Color.RED) {
    return this.viewer.entities.add({
      id,
      name,
      position: Cesium.Cartesian3.fromDegrees(lng, lat),
      point: { pixelSize: 10, color },
      label: {
        text: name,
        font: '14px sans-serif',
        fillColor: Cesium.Color.WHITE,
        outlineColor: Cesium.Color.BLACK,
        outlineWidth: 2,
        verticalOrigin: Cesium.VerticalOrigin.BOTTOM,
        pixelOffset: new Cesium.Cartesian2(0, -12)
      }
    });
  }

  remove(id) {
    this.viewer.entities.removeById(id);
  }

  clear() {
    this.viewer.entities.removeAll();
  }
}

const manager = new CityMarkerManager(viewer);
manager.add('bj', '北京', 116.40, 39.90, Cesium.Color.RED);

📋 今日知识点总结

  • 🎨 Entity 是 CesiumJS 中描述地球对象的高级抽象，由 id、name、position、orientation、图形等组成
  • 🎨 12 种图形类型：point、billboard、label、polyline、corridor、wall、polygon、ellipse、rectangle、box、cylinder、ellipsoid、plane
  • 🎨 0D 用于标注；1D 用于轨迹/管线/围栏；2D 用于区域覆盖；3D 用于立体对象
  • 🎨 材质决定外观：Color、Image、Checkerboard、Stripe、Grid、PolylineGlow、PolylineArrow 等
  • 🎨 动态属性：CallbackProperty 用于实时帧更新；SampledPositionProperty 用于时间轴动画
  • 🎨 EntityCollection：add / getById / remove / removeById / removeAll / values 管理实体
  • 🎨 最佳实践：给 Entity 设置 id，用 class 封装管理器，使用 Map 做外部索引
