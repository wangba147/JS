什么是 CesiumJS？为什么学它？

入门

🌎 CesiumJS 是什么？

CesiumJS 是一个开源的 JavaScript 库，用于创建三维地球和二维地图的 Web 应用。它基于 WebGL（你在 Day 12 学的 Canvas 的 3D 版本），可以在浏览器中直接渲染全球范围的高精度地形、卫星影像、3D 建筑模型、矢量数据——无需安装任何插件。

简单说：CesiumJS 就是浏览器里的 Google Earth，但它是给开发者用的——你可以完全控制它的一切。

🎯 用来做什么？（真实应用场景）

行业典型应用
航空航天卫星轨道可视化、火箭发射轨迹模拟、空间碎片追踪
智慧城市3D 建筑白模、BIM 模型叠加、城市交通监控
海洋/气象洋流可视化、台风路径、天气预报数据展示
军事/国防战场态势、雷达覆盖范围、导弹轨迹模拟
自动驾驶高精地图、传感器数据可视化、路径规划展示
物流/IoT实时车辆/船舶追踪、无人机航线规划
自然资源森林覆盖、矿产分布、地质灾害监测
💡
为什么选 CesiumJS 而不是 Three.js？Three.js 是通用 3D 引擎——适合做游戏、产品展示。CesiumJS 专门为地理空间数据设计——它内置了全球坐标系（WGS84）、地图投影、地形渲染、时间轴动画等 GIS 功能。如果你要做地球相关的应用，CesiumJS 省掉你 90% 的工作量。

🌐 Viewer — 一切的入口

new Cesium.Viewer('container') 是你创建的第一个对象。它像一个"管家"——包含场景、相机、地球、UI 控件、事件处理等一切。

const viewer = new Cesium.Viewer('cesiumContainer', {
  // 大量配置选项（后面会学）
  animation: false,       // 关闭底部动画控件
  timeline:  false,       // 关闭时间轴
  baseLayerPicker: false // 关闭底图选择器
});

// Viewer 的核心子对象：
viewer.scene;      // Cesium.Scene — 3D 场景，管理所有可视化内容
viewer.camera;     // Cesium.Camera — 相机，控制你看地球的视角
viewer.entities;   // Cesium.EntityCollection — 实体集合，你添加的所有东西
viewer.dataSources;// Cesium.DataSourceCollection — 数据源集合
viewer.imageryLayers;// 影像图层（卫星图/地图）

🎨 Scene — 3D 世界的舞台

Scene 管理所有被渲染的东西：Primitive（图元）、Globe（地球）、SkyBox（天空盒）、大气效果、光照。大多数时候你通过 Viewer 间接操作 Scene，但深入开发时需要直接访问它。

// Scene 的常用操作
viewer.scene.globe.enableLighting = true;   // 开启光照（有阴影）
viewer.scene.skyBox            // 天空盒（默认显示星空）
viewer.scene.sun               // 太阳光源
viewer.scene.moon              // 月球
viewer.scene.primitives        // 图元集合（性能优化的渲染对象）
viewer.scene.mode = Cesium.SceneMode.SCENE2D; // 切换到 2D 模式

🌎 Globe — 地球本身

Globe 代表地球的"壳"——它管理地表影像（卫星图）和地形高度。

// 访问 Globe
const globe = viewer.scene.globe;

// 常用属性
globe.show              = true;      // 显示/隐藏地球
globe.baseColor         = Cesium.Color.BLUE;  // 基础颜色（无影像时显示）
globe.depthTestAgainstTerrain = true; // 实体被地形遮挡

📷 Camera — 你的眼睛

Camera 决定你从哪个角度看地球。它有三个核心属性：位置（position）、朝向（direction/heading）、俯仰（pitch）。

// 方法 1：直接设置视角（瞬间跳转）
viewer.camera.setView({
  destination: Cesium.Cartesian3.fromDegrees(
    116.3974,  // 经度 — 北京
    39.9093,   // 纬度 — 北京
    15000      // 高度（米）
  ),
  orientation: {
    heading: Cesium.Math.toRadians(0),   // 方位角（北=0，顺时针）
    pitch:   Cesium.Math.toRadians(-45),// 俯仰角（负=向下看）
    roll:    0
  }
});

// 方法 2：飞行动画（3 秒飞过去）
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(-122.4194, 37.7749, 5000),
  duration: 3  // 秒
});
💡
记忆口诀：Viewer 是房子，Scene 是客厅，Globe 是地球仪，Camera 是你的眼睛，Entity 是你摆进去的家具。

核心对象：Entity / Primitive / DataSource

API

🎨 三者的区别

EntityPrimitiveDataSource
定位 高级 API，易用 低级 API，高性能 数据加载器
使用场景 快速添加点、线、面、模型（日常开发首选） 海量对象（百万级）需要极致性能 从 GeoJSON/CZML/KML 加载数据
渲染方式 内部转为 Primitive 直接 WebGL 渲染 内部创建 Entity
代码量 少 多 极简（一行）

🚀 Entity — 90% 时间都在用它

Entity 是 CesiumJS 最常用的 API。你可以把它理解为"地球上的一件东西"——一个点、一根线、一个多边形、一个 3D 模型。它有位置（position）和图形（point/polyline/polygon/model 等）。

// 添加一个红色的点标记
const entity = viewer.entities.add({
  id:       'my-point',       // 唯一标识（可选，方便后续查找/删除）
  name:     '北京',            // 显示名称
  position: Cesium.Cartesian3.fromDegrees(116.3974, 39.9093),
  point: {                    // 图形：点
    pixelSize: 10,           // 像素大小
    color:     Cesium.Color.RED,
    outlineColor: Cesium.Color.WHITE,
    outlineWidth: 2
  },
  label: {                    // 图形：文字标签
    text: '北京',
    font: '16px sans-serif',
    style: Cesium.LabelStyle.FILL_AND_OUTLINE,
    fillColor: Cesium.Color.WHITE,
    outlineColor: Cesium.Color.BLACK,
    outlineWidth: 3,
    verticalOrigin: Cesium.VerticalOrigin.BOTTOM,
    pixelOffset: new Cesium.Cartesian2(0, -14)  // 偏移，避免标签遮挡点
  }
});

// 查询实体
const found = viewer.entities.getById('my-point');

// 删除实体
viewer.entities.remove(entity);
💡
Entity 的图形（Graphics）类型：每个 Entity 可以同时附加多种图形——point（点）、polyline（线）、polygon（面）、model（3D 模型）、billboard（图标）、label（标签）、ellipse（椭圆）、rectangle（矩形）、corridor（走廊）、wall（墙）、box（立方体）……一个 Entity 可以同时有 position + point + label + polyline。

环境搭建：CDN + Cesium Ion Token

配置

🛠️ 方式一：CDN 引入（推荐入门）

最简单的入门方式——只需 CDN 引入 JS 和 CSS，无需构建工具：

<!-- 1. 引入 CesiumJS CSS -->
<link rel="stylesheet"
      href="https://cesium.com/downloads/cesiumjs/releases/1.121/Build/Cesium/Widgets/widgets.css">

<!-- 2. 引入 CesiumJS JavaScript -->
<script src="https://cesium.com/downloads/cesiumjs/releases/1.121/Build/Cesium/Cesium.js"></script>

<!-- 3. 指定地球容器 -->
<div id="cesiumContainer" style="width:100%;height:100vh"></div>

📦 方式二：npm 安装（推荐项目开发）

# 安装 CesiumJS
npm install cesium

# 然后在 JS 中 import
import * as Cesium from 'cesium';
import 'cesium/Build/Cesium/Widgets/widgets.css';

// 需要配置 vite/webpack 处理静态资源（后面会讲）

🔑 Cesium Ion Token — 必配！

从 CesiumJS 1.104+ 开始，默认底图（Bing Maps）和 Cesium World Terrain 地形需要 token。免费注册即可获得：

  1. 访问 https://ion.cesium.com 注册（免费）
  2. 进入 Access Tokens → 复制默认 token
  3. 在代码最前面设置：
// 必须放在 new Cesium.Viewer() 之前！
Cesium.Ion.defaultAccessToken =
  '你的 token 粘贴在这里';

const viewer = new Cesium.Viewer('cesiumContainer');
⚠️
没有 Token 的后果：地球会显示为纯色球体（没有卫星图），地形也会缺失。虽然可以切换到 OpenStreetMap 等免费底图——但地形渲染仍然需要 Ion。所以先注册，拿到 token，这是第一步。

Hello Earth：第一个 CesiumJS 程序

实战

🌎 最小化完整示例

创建一个 HTML 文件，复制以下代码，用浏览器打开就能看到旋转的地球：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>Hello CesiumJS!</title>
  <link rel="stylesheet"
        href="https://cesium.com/downloads/cesiumjs/releases/1.121/Build/Cesium/Widgets/widgets.css">
  <style>
    html, body, #cesiumContainer {
      width: 100%; height: 100%; margin: 0; padding: 0; overflow: hidden;
    }
  </style>
</head>
<body>
  <div id="cesiumContainer"></div>

  <script src="https://cesium.com/downloads/cesiumjs/releases/1.121/Build/Cesium/Cesium.js">
  </script>

  <script>
    // 1. 设置 Token（用自己的 token 替换！）
    Cesium.Ion.defaultAccessToken =
      '你的 Ion Token';

    // 2. 创建 Viewer
    const viewer = new Cesium.Viewer('cesiumContainer');

    // 3. (可选) 飞到中国上空
    viewer.camera.flyTo({
      destination: Cesium.Cartesian3.fromDegrees(
        104.0,  // 经度：中国中心
        35.0,   // 纬度：中国中心
        10000000 // 1 万公里高度（看到整个中国）
      ),
      duration: 3
    });

    console.log('🌎 地球已就绪！');
  </script>
</body>
</html>

🔧 Viewer 常用配置项

创建 Viewer 时可以传入第二个参数——一个配置对象，控制显示哪些默认 UI 控件：

const viewer = new Cesium.Viewer('cesiumContainer', {
  animation:       false,   // 隐藏底部动画控件
  timeline:        false,   // 隐藏时间轴
  baseLayerPicker: false,   // 隐藏底图选择器
  geocoder:        false,   // 隐藏地名搜索框
  homeButton:      false,   // 隐藏 Home 按钮
  sceneModePicker:  false,   // 隐藏 2D/3D 切换按钮
  navigationHelpButton: false, // 隐藏帮助按钮
  fullscreenButton: false,   // 隐藏全屏按钮
  infoBox:         false,   // 隐藏信息框（点击实体默认弹出）
  selectionIndicator: false, // 隐藏选择指示器
  // 也可以保留部分，按需选择
});
💡
开发阶段建议：先全部 false，自己通过代码控制一切。等熟悉了再根据需要打开某些控件。

底图与地形：ImageryLayer & Terrain

地图

🎨 底图（ImageryLayer）— 地球的"皮肤"

底图是覆盖在地球表面的影像——卫星图、街道图、地形图等。你可以叠加多层底图，上层透明的地方会露出下层。

// 方法 1：创建时指定（替换默认 Bing Maps）
const viewer = new Cesium.Viewer('cesiumContainer', {
  imageryProvider: new Cesium.UrlTemplateImageryProvider({
    url: 'https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png'
  })
});

// 方法 2：动态添加/替换图层
const layer = viewer.imageryLayers.addImageryProvider(
  new Cesium.UrlTemplateImageryProvider({
    url: 'https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png'
  })
);

⛰️ 常用底图源一览

底图Provider 类说明
Bing Maps BingMapsImageryProvider 默认底图，有卫星/道路/混合模式，需 token
OpenStreetMap OpenStreetMapImageryProvider 免费，街道图，初学者友好
ArcGIS ArcGisMapServerImageryProvider Esri 提供的全球影像
天地图（中国） UrlTemplateImageryProvider 中国地区精度更高，需申请 key
自定义瓦片 UrlTemplateImageryProvider 任意 XYZ/TMS 瓦片服务
单张图片 SingleTileImageryProvider 覆盖全球的一张图

⛰️ 地形（Terrain）— 地球的"骨架"

地形决定了地球表面的起伏——从平面球体变成有山脉和峡谷的真实地球。

// 添加 Cesium World Terrain（全球高精度地形，需要 Ion Token）
viewer.terrainProvider = await Cesium.createWorldTerrainAsync({
  requestWaterMask: true,     // 请求水面遮罩（海洋有光泽效果）
  requestVertexNormals: true // 请求法线（更好的光照效果）
});

// 无地形的默认模式（平滑球体）
viewer.terrainProvider = new Cesium.EllipsoidTerrainProvider();
💡
多图层叠加：viewer.imageryLayers 是一个有序的图层列表。你可以同时显示卫星图 + 行政边界 + 交通路网——底层是卫星图，上面叠半透明的边界和路网。这使得 CesiumJS 远超普通二维地图。

相机控制：飞到地球上任何一个角落

视角

✈️ 四种移动相机的方式

方式API特点
瞬间跳转 camera.setView() 立即切换视角，无动画
飞行 camera.flyTo() 带弧线动画飞过去，推荐
缩放 camera.zoomIn() / zoomOut() 基于当前位置缩放
看向某点 camera.lookAt() 保持当前位置，朝向某点

🚀 flyTo 的完整参数

viewer.camera.flyTo({
  // destination：飞到哪里
  destination: Cesium.Cartesian3.fromDegrees(
    121.4737,  // 经度 — 上海
    31.2304,   // 纬度 — 上海
    3000       // 高度（米）
  ),

  // orientation：以什么姿态看（可选）
  orientation: {
    heading: Cesium.Math.toRadians(90),   // 朝东看
    pitch:   Cesium.Math.toRadians(-60),// 向下 60 度
    roll:    0
  },

  duration: 3,           // 飞行时长（秒）
  easingFunction: Cesium.EasingFunction.QUADRATIC_IN_OUT
});

// 常用高度参考：
// 1000m     — 城市级（俯视城区）
// 10000m    — 省级（看到整个省份）
// 100000m   — 国家级
// 10000000m — 全球视角
// 25000000m — 能看到半个地球

鼠标控制（无需代码）：左键拖拽=旋转地球、右键拖拽=缩放、中键拖拽=平移。这些是 Viewer 自带的交互，你的用户不需要写一行代码就能浏览全球。

添加第一个 Entity：点、线、标注

实战

📍 综合示例：在地球上标记五个中国城市

// 中国五大城市坐标
const cities = [
  { name: '北京',   lng: 116.3974, lat: 39.9093,  color: Cesium.Color.RED },
  { name: '上海',   lng: 121.4737, lat: 31.2304,  color: Cesium.Color.BLUE },
  { name: '广州',   lng: 113.2644, lat: 23.1291,  color: Cesium.Color.GREEN },
  { name: '成都',   lng: 104.0665, lat: 30.5728,  color: Cesium.Color.YELLOW },
  { name: '乌鲁木齐', lng: 87.6168,  lat: 43.8256,  color: Cesium.Color.ORANGE }
];

cities.forEach((city, index) => {
  viewer.entities.add({
    name: city.name,
    position: Cesium.Cartesian3.fromDegrees(city.lng, city.lat),
    point: {
      pixelSize: 10,
      color: city.color
    },
    label: {
      text: city.name,
      font: '14px sans-serif',
      fillColor: Cesium.Color.WHITE,
      outlineColor: Cesium.Color.BLACK,
      outlineWidth: 2,
      style: Cesium.LabelStyle.FILL_AND_OUTLINE,
      verticalOrigin: Cesium.VerticalOrigin.BOTTOM,
      pixelOffset: new Cesium.Cartesian2(0, -12)
    }
  });
});

// 飞到能看见所有城市的视角
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(104.0, 35.0, 5000000)
});

📏 Entity 的更多图形类型预览

// 线 (Polyline)
viewer.entities.add({
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([
      116.39, 39.91,   // 北京
      121.47, 31.23    // 上海
    ]),
    width: 3,
    material: Cesium.Color.CYAN
  }
});

// 多边形 (Polygon)
viewer.entities.add({
  polygon: {
    hierarchy: Cesium.Cartesian3.fromDegreesArray([
      116.0, 39.5,
      117.0, 39.5,
      117.0, 40.5,
      116.0, 40.5
    ]),
    material: Cesium.Color.RED.withAlpha(0.3),
    outline: true,
    outlineColor: Cesium.Color.RED
  }
});

✍️ 今日练习（3 道）

📋 今日知识点总结

  • 🌎 CesiumJS 是基于 WebGL 的开源三维地球引擎，用来在浏览器中创建地理可视化应用
  • 🌎 核心架构：Viewer（总入口）→ Scene（3D 场景）+ Camera（视角）+ Globe（地球）+ ImageryLayers（图层）
  • 🌎 Entity vs Primitive：Entity 是高级 API（易用，90% 场景足够）；Primitive 是低级 API（高性能，海量数据场景）
  • 🌎 Viewer 配置：创建时可通过第二个参数控制 UI 控件显隐（animation/timeline/baseLayerPicker 等 10+ 开关）
  • 🌎 底图：默认 Bing Maps（需 Token）；可切换为 OpenStreetMap（免费）或其他地图瓦片服务
  • 🌎 地形：Cesium World Terrain 提供全球高精度地形；createWorldTerrainAsync() 异步加载
  • 🌎 相机：setView() 瞬间跳转；flyTo() 带动画飞行（最常用）；高度单位是米
  • 🌎 Entity 添加：viewer.entities.add({ position, point/label/polyline/polygon/model... })
  • 🌎 Cesium Ion Token：使用 Cesium 服务的前置条件（底图/地形/3D Tiles），免费注册即可获得
  • 🌎 坐标系：Cartesian3.fromDegrees(lng, lat, height) 将经纬度转为 Cesium 内部坐标
