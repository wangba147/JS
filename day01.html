变量声明：var、let、const

VARIABLES

变量是用来存储数据的命名容器。JavaScript 有三个关键字用于声明变量：

关键字 可重新赋值 块级作用域 建议
var ✅ 是 ❌ 否（函数作用域） ⚠️ 旧式，避免使用
let ✅ 是 ✅ 是 ✅ 推荐（可变量）
const ❌ 否 ✅ 是 ✅ 推荐（常量/对象） 
// ① let — 可重新赋值的变量
let longitude = 116.4;    // 经度（北京）
longitude = 121.5;        // ✅ 可以重新赋值（上海）

// ② const — 不可重新赋值
const EARTH_RADIUS = 6371; // 地球半径（km），不会变
// EARTH_RADIUS = 7000;   ❌ TypeError: 不能重新赋值

// ③ 块级作用域演示
{
  let inside = "块内变量";
  console.log(inside);  // ✅ "块内变量"
}
// console.log(inside); ❌ ReferenceError: inside 未定义

// ④ 同时声明多个变量
let x = 10, y = 20, z = 30;
💡
最佳实践：默认用 const，只有当你确定需要修改这个变量时，才改用 let。这能让代码意图更清晰。
2

基本数据类型（7 种）

TYPES

JavaScript 有 7 种原始数据类型（Primitive Types）：

🔢 Number — 数字

整数和浮点数统一为一种类型。

let lat  = 39.9042;  // 浮点
let zoom = 15;      // 整数
let inf  = Infinity; // 无穷大
let nan  = NaN;      // 非数字

📝 String — 字符串

用单引号、双引号或模板字符串（反引号）表示。

let city = '北京';
let info = `城市：${city}`;
// 模板字符串可嵌入变量 ↑

✅ Boolean — 布尔

只有两个值：true 或 false。

let isVisible = true;
let hasLoaded = false;

❓ Undefined — 未定义

变量声明但未赋值时，默认为 undefined。

let altitude;
console.log(altitude); // undefined

🚫 Null — 空值

表示"有意为空"，是开发者主动赋的空值。

let viewer = null;
// 稍后再初始化 Cesium Viewer

🔑 Symbol — 符号

ES6 新增，创建唯一标识符，用于对象属性键。

const id = Symbol('layerId');
// 每个 Symbol 值都是唯一的

🧱 BigInt — 大整数

处理超过 Number 安全范围（±2⁵³）的整数，在数字末尾加 n。

const bigNum = 9007199254740993n; // 精确大整数
console.log(typeof bigNum);       // "bigint"
ℹ️
除了以上 7 种原始类型，JavaScript 还有一种引用类型：Object（包括数组、函数等）。引用类型将在 Day 03 深入学习。
3

类型检测与转换

TYPE SYSTEM

🔍 typeof 运算符

用于检测一个值的类型，返回字符串。

console.log(typeof 42);          // "number"
console.log(typeof "hello");    // "string"
console.log(typeof true);        // "boolean"
console.log(typeof undefined);  // "undefined"
console.log(typeof null);       // "object"  ⚠️ JS 历史遗留 bug
console.log(typeof {});          // "object"
console.log(typeof []);          // "object"  ⚠️ 数组也是 object
console.log(typeof function(){}); // "function"

🔄 类型转换

JavaScript 是动态类型语言，支持显式和隐式转换：

// ① 转数字
Number("123")    // → 123
Number("12.5")  // → 12.5
Number(true)    // → 1
Number(false)   // → 0
Number("")      // → 0
Number("abc")   // → NaN
parseInt("42px") // → 42（解析整数部分）
parseFloat("3.14m") // → 3.14

// ② 转字符串
String(42)      // → "42"
(42).toString() // → "42"
`值是 ${42}`    // → "值是 42"

// ③ 转布尔
Boolean(0)      // → false（假值）
Boolean("")     // → false（假值）
Boolean(null)   // → false（假值）
Boolean(1)      // → true
Boolean("hi")  // → true
⚠️
假值（Falsy）记忆口诀：false、0、""（空字符串）、null、undefined、NaN — 这 6 个值转布尔都为 false，其余所有值均为 true。

算术与赋值运算符

OPERATORS

➕ 算术运算符

let a = 10, b = 3;

a + b   // 13  加法
a - b   // 7   减法
a * b   // 30  乘法
a / b   // 3.333... 除法
a % b   // 1   取余（模）
a ** b  // 1000 幂运算

// 自增 / 自减
a++     // 先用再加（a=10）
++a     // 先加再用（a=12）
a--     // 先用再减

📌 赋值运算符

let n = 10;

n += 5;   // n = n+5 → 15
n -= 3;   // n = n-3 → 12
n *= 2;   // n = n*2 → 24
n /= 4;   // n = n/4 → 6
n %= 4;   // n = n%4 → 2
n **= 3;  // n = n**3 → 8

// 解构赋值（ES6+）
let [x, y, z] = [1, 2, 3];

🌍 CesiumJS 场景：计算地图比例尺

将来在 CesiumJS 中，你会大量用到算术运算符来计算坐标、高度、角度等：

// 示例：将度数转换为弧度（CesiumJS 常用）
const PI = Math.PI;
const toRadians = (deg) => deg * PI / 180;

console.log(toRadians(90));   // 1.5707... (π/2)
console.log(toRadians(180));  // 3.1415... (π)

比较与逻辑运算符

LOGIC

⚖️ 比较运算符

// == vs ===  这是 JS 最重要的区分之一！
5 ==  "5"   // true  ← 宽松比较（自动类型转换）
5 === "5"   // false ← 严格比较（类型 + 值都要相同）✅ 推荐

5 !=  "5"   // false（宽松不等）
5 !== "5"   // true （严格不等）✅ 推荐

10 >  5    // true
10 >= 10   // true
3  <  7    // true
3  <= 3    // true

🔗 逻辑运算符

// && (AND) — 两个都为 true，结果才为 true
true  && true   // true
true  && false  // false

// || (OR) — 有一个为 true，结果就为 true
false || true   // true
false || false  // false

// ! (NOT) — 取反
!true            // false
!false           // true

// ?? (空值合并，ES2020) — 左侧为 null/undefined 时取右值
const zoom = null ?? 15;  // 15
const name = "Alice" ?? "默认"; // "Alice"

// 短路求值 — 常用技巧
const user = null;
const name2 = user && user.name; // null（不会报错）
💡
永远用 === 而不是 ==！ 宽松比较会引发各种隐式转换的 bug，是 JavaScript 初学者最常踩的坑。

字符串操作

STRINGS

✂️ 常用字符串方法

const str = "  Hello, CesiumJS!  ";

// 长度
str.length            // 20

// 大小写
str.toUpperCase()      // "  HELLO, CESIUMJS!  "
str.toLowerCase()      // "  hello, cesiumjs!  "

// 去除首尾空格
str.trim()             // "Hello, CesiumJS!"

// 查找
str.includes("Cesium")  // true
str.indexOf("Cesium")   // 9（起始索引）
str.startsWith("  H") // true
str.endsWith("!  ")   // true

// 截取
str.slice(2, 7)         // "Hello"
str.slice(-3)           // "!  "（从末尾数）

// 替换
str.replace("Hello", "Hi") // "  Hi, CesiumJS!  "

// 分割
"116.4,39.9".split(",")  // ["116.4", "39.9"]

// 模板字符串（ES6+）
const lon = 116.4, lat = 39.9;
console.log(`坐标：经度 ${lon}°，纬度 ${lat2}°`);
// 输出：坐标：经度 116.4°，纬度 39.9°

📋 今日知识点总结

  • ✅ const（不可重赋值）和 let（可重赋值）是现代 JS 变量声明标准，避免使用 var
  • ✅ JS 有 7 种原始类型：Number、String、Boolean、Undefined、Null、Symbol、BigInt
  • ✅ 用 typeof 检测类型；注意 typeof null === "object" 是历史 bug
  • ✅ 6 个假值：false、0、""、null、undefined、NaN
  • ✅ 始终使用 ===（严格相等），而非 ==（宽松相等）
  • ✅ 模板字符串（反引号）是格式化字符串的最优雅方式
